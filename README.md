# Éblouissant — Website

The single-page website for **Éblouissant / Anuragini Pandey, MD** — a board-certified physician and aesthetic practice in Greenwich Village, NYC (31 Washington Square West, Suite 3F).

> Note: the practice email is still `info@integramedicalnyc.com`. The email domain hasn't been migrated to the new Éblouissant name yet, so that address stays as-is for now.

- **`index.html`** — the entire website. It's one self-contained file (HTML + CSS + a little JavaScript, all inline). Open it in any web browser to preview it.
- **`dr-anu-cropped.jpg`** — Dr. Pandey's photo, used in the About section.

## To preview it

Double-click `index.html` — it opens in your web browser. That's the whole site.

## To make changes

You don't need to know how to code. Open the [Claude desktop app](https://claude.ai/download), point it at this repository, and describe what you want changed in plain English (for example: *"update my office hours to open at 9am"*). See `CLAUDE.md` for the kinds of edits that are easy to make and where they live.

## Design

A soft "watercolour" blush-and-gold theme. Fonts: Cormorant Garamond (headings) + DM Sans (body). Booking runs through [Zocdoc](https://www.zocdoc.com/doctor/anuragini-pandey-md-567010).
